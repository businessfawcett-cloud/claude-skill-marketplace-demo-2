# claude-skill-marketplace-demo-2

POC marketplace for testing whether Claude Code can load multiple plugins from
one marketplace simultaneously in a single consuming repo. Three plugins
(`skill-alpha`, `skill-bravo`, `skill-charlie`), each with one skill, no version
pinning (commit-SHA versioning).

See `claude-marketplace-consumer-demo-2` and
`D:\Internship\Marketplace 2\FINDINGS.md` for the test setup and results.
